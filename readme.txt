List and export product quantity report by each order at one place.

== Description ==

Admin can filter by <strong>date range</strong> and <strong>order status</strong>, search results will 
generate as per filter one can export report as <strong>CSV</strong>.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/product-wise-order-export-for-woocommerce-by-vk.zip` directory, or install the plugin through the WordPress plugins screen directly.<br />

2. Activate the "Product Quantity Report By Order for Woocommerce" plugin through the 'Plugins' screen in WordPress.<br />

3. Use the "Dashboard > Order Wise Product Quantity" menu in admin screen to filter and export reports
